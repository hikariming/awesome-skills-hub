# Contributing to Awesome Skills Hub

Thank you for your interest in contributing to Awesome Skills Hub!

## How to Contribute

### Adding a New Skill

If you've created an amazing Claude AI skill that you'd like to share:

1. **Visit SkillsHub** - Go to [skillshub.dev](https://skillshub.dev) and submit your skill
2. **Update This List** - Once your skill is added to SkillsHub, it will automatically appear in this list when we regenerate

### Reporting Issues

Found a mistake or have a suggestion? Please:
- Open an issue on GitHub
- Or contact us at [skillshub.dev](https://skillshub.dev)

### Updating the List

The README.md is automatically generated from the SkillsHub database. To update it:

```bash
npm run generate
```

This will fetch the latest skills from SkillsHub and regenerate the README.md file.

## Code of Conduct

Be respectful, inclusive, and constructive. We're all here to help each other build amazing AI tools!
